EN:
Take Note is a minimalist offline-first note app built for quick

With the block editor, you can combine:

- Text
- Headings and subtitles
- Quotes
- Bullet lists
- Numbered lists
- Checklists
- Images
- In-app audio recording (playback at 1x, 1.5x, and 2x)

Main features:

- Block options accessible from "Aa" and also from a fixed editor button
- Note search
- Category organization with custom colors

- Archive and unarchive notes
